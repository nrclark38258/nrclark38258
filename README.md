# Hi, I'm Nathan Clark

Data scientist and analytics architect based in Maine. My work lives at the intersection of data engineering, statistical analysis, and cross-functional problem solving — building the infrastructure that makes complex systems measurable and the tooling that makes data accessible to the people who need it.

## What I do

- **Pipeline architecture** — event-driven and scheduled AWS data pipelines (Lambda, S3, SNS, SQS, EventBridge) processing real-world instrument and telemetry data at scale
- **Analytics infrastructure** — Snowflake data warehouse design, multi-source operational dashboards, automated reporting pipelines
- **Metric development** — defining what "good" looks like for complex systems, building the monitoring to measure it continuously, and conducting causal analysis when it isn't
- **Self-service tooling** — analytics tools and scripted automation that eliminate manual work and put data in the hands of non-technical stakeholders
- **CI/CD for analytical tooling** — automated deployment pipelines for analytics software, including a GitHub Actions-based system presented at JMP Discovery Americas Conference (2023 & 2025)
